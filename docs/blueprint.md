# **App Name**: Divine Compass

## Core Features:

- Dynamic Bible Reader: Seamlessly browse and display Bible passages with book, chapter, and verse navigation, powered by the YouVersion API.
- Secure User Authentication: User registration and login, including profile management, utilizing Firebase Authentication and Firestore for secure access and profile storage.
- Personal Verse Interactions: Users can bookmark and highlight their favorite verses for easy access and study, with data stored in Firestore.
- AI Scripture Assistant Tool: An intelligent AI tool that provides insights, answers questions, or generates reflections on Bible passages using the Gemini API.
- Private Prayer Journal: Enable users to create, view, and manage their personal prayer entries, securely stored in Firestore.
- Guided Reading Plans: Access a curated selection of Bible reading plans to help users engage with scripture daily, with basic tracking capabilities.
- Admin Audio Content Manager: An administrative interface for uploading, organizing, and managing audio resources (e.g., podcasts, sermons) associated with Bible content, stored in Firebase.

## Style Guidelines:

- Primary Color: A deep, spiritual sapphire blue (#1A60E5) to establish a premium and authoritative yet calming presence, offering strong contrast against dark backgrounds.
- Background Color: An extremely dark, subtly bluish-grey (#0B0C0D) for the main backdrop, fulfilling the 'dark mode default (bg-zinc-950)' request with a sophisticated feel.
- Accent Color: Vibrant `emerald-500` (#07BF7A) to highlight interactive elements, calls to action, and convey a sense of 'holy green' energy as specified.
- Verse/Headline Font: 'Playfair Display' (serif) for Bible verses and prominent headlines, lending an elegant, traditional, and modern aesthetic as specifically requested by the user.
- Body/UI Font: 'Inter' (sans-serif) for all general user interface elements, navigation text, and supplementary information, ensuring clarity and a clean, contemporary user experience.
- Modern and clean line icons from 'lucide-react' will be used for interactive elements and the bottom navigation, aligning with a YouVersion-like minimal and intuitive visual style.
- The layout will default to a dark theme (`bg-zinc-950` equivalent) with a prominent, intuitive bottom navigation bar and organized content sections, consistent with a clean YouVersion-like design.
- Subtle and smooth animated transitions and interactive effects, powered by `framer-motion`, to enhance the premium feel and user feedback throughout the application.
